# MixMusic

一个插件化的音乐播放器

修改 Model 之后 运行 ```flutter pub run build_runner build --delete-conflicting-outputs```

## 说明

该软件不内置任何音源，一切播放能力基于js插件

该软件处于试验阶段，一切api都有可能改变

代码很垃圾，就实现了基本功能

- [x] 播放
- [x] 歌单分类，列表 ，详情
- [x] 专辑分类，列表，详情
- [x] 搜索
- [x] 跨站点匹配
- [x] 歌手
- [x] 榜单
- [x] 新歌

- [x] 首页自定义数据
